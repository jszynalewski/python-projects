# Python Projects Portfolio

Kolekcja moich projektów programistycznych zrealizowanych w języku Python. Repozytorium stanowi odzwierciedlenie mojego rozwoju technicznego, nauki dobrych praktyk oraz praktycznego podejścia do rozwiązywania problemów i zarządzania danymi.

## Spis projektów

### 1. [Telegram Bot](./botTelegram.py)
W pełni funkcjonalny bot stworzony na platformę Telegram, demonstrujący umiejętność pracy z zewnętrznymi API oraz obsługę zapytań użytkowników.
* **Kluczowe zagadnienia:** Integracja z API Telegrama, podstawy bezpieczeństwa danych.

### 2. [CRUD Application](./crud_app.py)
Aplikacja realizująca podstawowe operacje na danych: Create, Read, Update, Delete. Projekt udowadnia zrozumienie cyklu życia danych oraz umiejętność ich przetwarzania i modyfikowania.
* **Kluczowe zagadnienia:** Architektura aplikacji, bazy danych (SQLite), walidacja danych wejściowych.

### 3. [System Biblioteki - OOP](./biblioteka.py)
Aplikacja do zarządzania zbiorami bibliotecznymi, zaprojektowana w oparciu o paradygmat programowania obiektowego (Object-Oriented Programming).
* **Kluczowe zagadnienia:** Klasy, dziedziczenie, enkapsulacja, modelowanie procesów biznesowych i tworzenie czytelnego kodu.

## Wykorzystane Technologie
* **Języki i narzędzia:** Python 3.x, SQLite
* **Wersjonowanie:** Git, GitHub
